# Weather App

A simple and responsive Weather App built with **HTML, CSS, and JavaScript** that fetches real-time weather data using the **Open-Meteo API**.

## Features

- Search weather by city name
- Fetches latitude and longitude using Geocoding API
- Displays current temperature and wind speed
- Handles invalid city input
- Clean and responsive UI
- Dynamic weather card rendering using JavaScript

## Tech Stack

- HTML
- CSS
- JavaScript
- Open-Meteo Geocoding API
- Open-Meteo Weather API

## How It Works

1. User enters a city name
2. App fetches latitude and longitude for that city
3. Using those coordinates, the app fetches current weather data
4. Weather details are displayed dynamically on the page

## APIs Used

### Geocoding API
Used to get latitude and longitude from a city name.

### Weather API
Used to fetch current weather data using latitude and longitude.
